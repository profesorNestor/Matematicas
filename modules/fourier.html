🎨 El Lienzo Interactivo

El lienzo es tu área de juegos. Aquí puedes manipular la visualización de las funciones:

  • 🖱️ Arrastrar (Pan): Haz clic y arrastra con el mouse para moverte por el plano cartesiano.
  • 휠 Zoom: Usa la rueda del mouse para acercar o alejar la vista.
  • ↔️ Escalar Eje X: Acerca el cursor al eje Y (la línea vertical) hasta que cambie a ↔️, luego arrastra para estirar o encoger el eje horizontal.
  • ↕️ Escalar Eje Y: Acerca el cursor al eje X (la línea horizontal) hasta que cambie a ↕️, luego arrastra para estirar o encoger el eje vertical.
  • 🖐️ Gestos Táctiles: En dispositivos móviles, usa un dedo para arrastrar (pan), y dos dedos para pellizcar (zoom). ¡También puedes arrastrar los ejes para escalarlos!
⚙️ Panel de Control
  • Función f(x): ¡El corazón de todo! Escribe aquí tu función. Puedes usar operadores (+, -, *, /, ^), constantes como pi y e, y funciones como sin(x), cos(x), abs(x), y sqrt(x). También puedes usar condiciones ternarias para funciones a trozos, por ejemplo: x > 0 ? x : -x para `abs(x)`.
  • Límite del Período (L): Define el intervalo [-L, L] sobre el cual se analiza la función. La longitud total del período es 2L. Un valor común es pi (3.14159...).
  • Número de Armónicos (N): ¿Cuántas ondas (senos y cosenos) quieres sumar para aproximar tu función? A mayor número, más precisa será la aproximación, ¡pero el cálculo será más intenso!
🤔 ¿Qué es una Serie de Fourier?

¡Es una idea matemática poderosa y fascinante! Imagina que cualquier función periódica, sin importar cuán extraña o compleja sea, puede ser descompuesta y reconstruida sumando un montón de ondas simples (senos y cosenos).

Esta herramienta calcula cuáles son esas ondas base (los armónicos) y qué tan "importante" o "fuerte" es cada una (los coeficientes an y bn). Al sumarlas todas, obtenemos una aproximación de la función original. ¡Es como tener un set infinito de bloques LEGO para construir cualquier forma de onda!
