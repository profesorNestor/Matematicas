Formulario Matemático Definitivo

Álgebra y Álgebra Lineal

Productos Notables y Ecuaciones

  • Binomio al cuadrado: $(a \pm b)^2 = a^2 \pm 2ab + b^2$
  • Diferencia de cuadrados: $(a+b)(a-b) = a^2 - b^2$
  • Binomio al cubo: $(a \pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3$
  • Ecuación Cuadrática: Para $ax^2+bx+c=0$, $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$

Álgebra Lineal

  • Determinante 3x3: $\det\begin{pmatrix} a & b & c \\ d & e & f \\ g & h & i \end{pmatrix} = a(ei-fh) - b(di-fg) + c(dh-eg)$
  • Inversa de una Matriz: $A^{-1} = \frac{1}{\det(A)} (\text{adj}(A))^T$
  • Regla de Cramer (3x3): $x = \frac{\det(A_x)}{\det(A)}, y = \frac{\det(A_y)}{\det(A)}, z = \frac{\det(A_z)}{\det(A)}$
  • Valores Propios (Eigenvalores): Se obtienen de la ecuación característica $\det(A - \lambda I) = 0$.
Números Complejos

Operaciones y Fórmulas Fundamentales

  • Forma Rectangular: $z = a + bi$
  • Forma Polar: $z = r(\cos\theta + i\sin\theta) = r\text{cis}(\theta)$
  • Módulo: $|z| = r = \sqrt{a^2+b^2}$
  • Conjugado: $\bar{z} = a - bi$
  • Fórmula de Euler: $e^{i\theta} = \cos\theta + i\sin\theta$
  • Fórmula de De Moivre: $z^n = r^n(\cos(n\theta) + i\sin(n\theta))$
Geometría y Trigonometría

Áreas, Perímetros y Volúmenes

Figuras Planas

  • Triángulo: Área = $\frac{1}{2}bh$. Perímetro = $a+b+c$.
  • Rectángulo: Área = $lw$. Perímetro = $2l+2w$.
  • Círculo: Área = $\pi r^2$. Perímetro (Circunferencia) = $2\pi r$.
  • Polígono Regular (n lados, lado s): Área = $\frac{1}{4}ns^2\cot(\frac{\pi}{n})$. Perímetro = $ns$.

Sólidos

  • Esfera: Volumen = $\frac{4}{3}\pi r^3$. Área superficial = $4\pi r^2$.
  • Cilindro: Volumen = $\pi r^2 h$. Área superficial = $2\pi rh + 2\pi r^2$.
  • Cono: Volumen = $\frac{1}{3}\pi r^2 h$. Área superficial = $\pi r(r + \sqrt{h^2+r^2})$.
  • Cubo: Volumen = $a^3$. Área superficial = $6a^2$.

Geometría Analítica Plana

  • Distancia $P_1(x_1, y_1)$ a $P_2(x_2, y_2)$: $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
  • Ecuación punto-pendiente: $y - y_1 = m(x - x_1)$
  • Distancia de $P(x_0, y_0)$ a recta $Ax+By+C=0$: $d = \frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$
Combinatoria, Estadística y Probabilidad

Combinatoria y Progresiones

  • Permutaciones lineales: $P_n = n!$
  • Permutaciones circulares: $PC_n = (n-1)!$
  • Combinaciones sin repetición: $C_{n,k} = \binom{n}{k} = \frac{n!}{k!(n-k)!}$
  • Progresión Aritmética (suma): $S_n = \frac{n}{2}(a_1 + a_n)$
  • Progresión Geométrica (suma infinita): $S = \frac{a_1}{1-r}$ para $|r|<1$

Estadística Descriptiva

  • Regla de Sturges: $k = 1 + 3.322 \log(N)$
  • Media (agrupados): $\bar{x} = \frac{\sum f_i \cdot x_i}{N}$
  • Varianza (muestral): $s^2 = \frac{\sum (x_i - \bar{x})^2}{n-1}$
  • Regresión Lineal: $y = a + bx$, con $b = r \frac{s_y}{s_x}$, $a = \bar{y} - b\bar{x}$

Probabilidad

  • Probabilidad Condicional: $P(A|B) = \frac{P(A \cap B)}{P(B)}$
  • Probabilidad Total: $P(B) = \sum_{i=1}^{n} P(B|A_i)P(A_i)$
  • Teorema de Bayes: $P(A_i|B) = \frac{P(B|A_i)P(A_i)}{P(B)}$

Tabla Extendida de Integrales

  • $\int \tan(u) \,du = -\ln|\cos(u)| + C$
  • $\int \sec(u) \,du = \ln|\sec(u) + \tan(u)| + C$
  • $\int \frac{du}{\sqrt{a^2-u^2}} = \arcsin\left(\frac{u}{a}\right) + C$
  • $\int \frac{du}{u^2+a^2} = \frac{1}{a} \arctan\left(\frac{u}{a}\right) + C$
  • $\int \sqrt{a^2-u^2} \,du = \frac{u}{2}\sqrt{a^2-u^2} + \frac{a^2}{2}\arcsin\left(\frac{u}{a}\right) + C$
  • $\int e^{au}\sin(bu) \,du = \frac{e^{au}}{a^2+b^2}(a\sin(bu)-b\cos(bu)) + C$
Ecuaciones Diferenciales

Ecuaciones de Primer Orden

  • Lineales ($y' + P(x)y = Q(x)$): Factor integrante $\mu(x) = e^{\int P(x)dx}$. Solución: $y\mu(x) = \int \mu(x)Q(x)dx + C$.
  • Homogéneas ($y' = f(y/x)$): Sustitución $v = y/x$.
  • Exactas ($M(x,y)dx + N(x,y)dy = 0$): Si $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$.

Ecuaciones Lineales de 2º Orden

Para $ay'' + by' + cy = 0$, se resuelve la ecuación característica $ar^2+br+c=0$.
  • Raíces reales distintas $r_1, r_2$: $y_h = c_1e^{r_1x} + c_2e^{r_2x}$
  • Raíces reales iguales $r_1=r_2=r$: $y_h = (c_1 + c_2x)e^{rx}$
  • Raíces complejas $\alpha \pm i\beta$: $y_h = e^{\alpha x}(c_1\cos(\beta x) + c_2\sin(\beta x))$
