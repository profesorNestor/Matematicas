Álgebra

Productos Notables y Factorización

  • $(a \pm b)^2 = a^2 \pm 2ab + b^2$
  • $(a+b)(a-b) = a^2 - b^2$
  • $(a \pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3$
  • $a^3 \pm b^3 = (a \pm b)(a^2 \mp ab + b^2)$
  • $(a+b+c)^2 = a^2+b^2+c^2+2ab+2ac+2bc$
  • $x^n - y^n = (x-y)(x^{n-1} + x^{n-2}y + \dots + y^{n-1})$
  • $x^n + y^n = (x+y)(x^{n-1} - x^{n-2}y + \dots + y^{n-1})$ (si n es impar)

Logaritmos

  • Definición: $y = \log_b(x) \iff b^y = x$
  • Producto: $\log_b(xy) = \log_b(x) + \log_b(y)$
  • Cociente: $\log_b(x/y) = \log_b(x) - \log_b(y)$
  • Potencia: $\log_b(x^n) = n \log_b(x)$
  • Cambio de base: $\log_b(x) = \frac{\log_c(x)}{\log_c(b)}$

Binomio de Newton

$$(a+b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k$$ Donde el coeficiente binomial es: $\binom{n}{k} = \frac{n!}{k!(n-k)!}$
Lógica y Conjuntos

Leyes de la Lógica Proposicional

  • Idempotencia: $p \lor p \equiv p$ ; $p \land p \equiv p$
  • Asociativa: $(p \lor q) \lor r \equiv p \lor (q \lor r)$
  • Conmutativa: $p \lor q \equiv q \lor p$
  • Distributiva: $p \lor (q \land r) \equiv (p \lor q) \land (p \lor r)$
  • Leyes de De Morgan: $\neg(p \land q) \equiv \neg p \lor \neg q$
  • Implicación: $p \rightarrow q \equiv \neg p \lor q$

Leyes de los Conjuntos

  • Unión: $A \cup B = \{x \mid x \in A \text{ o } x \in B\}$
  • Intersección: $A \cap B = \{x \mid x \in A \text{ y } x \in B\}$
  • Leyes de De Morgan: $(A \cup B)^c = A^c \cap B^c$
Geometría y Trigonometría

Geometría Plana

  • Teorema de Pitágoras: $a^2 + b^2 = c^2$.
  • Ángulo central: $\alpha = \text{arco } AB$
  • Ángulo inscrito: $\alpha = \frac{1}{2}\text{arco } BC$
  • Ángulo interior: $\alpha = \frac{1}{2}(\text{arco } AB + \text{arco } CD)$
  • Ángulo exterior: $\alpha = \frac{1}{2}(\text{arco } AB - \text{arco } CD)$

Identidades Trigonométricas

  • $\sin^2(\theta) + \cos^2(\theta) = 1$
  • $1 + \tan^2(\theta) = \sec^2(\theta)$
  • Suma/Resta: $\sin(A \pm B) = \sin(A)\cos(B) \pm \cos(A)\sin(B)$
  • $\cos(A \pm B) = \cos(A)\cos(B) \mp \sin(A)\sin(B)$
  • Ángulo Doble: $\sin(2\theta) = 2\sin(\theta)\cos(\theta)$
  • $\cos(2\theta) = \cos^2(\theta) - \sin^2(\theta)$
  • Ángulo Mitad: $\sin^2\left(\frac{\theta}{2}\right) = \frac{1 - \cos(\theta)}{2}$

Leyes para Triángulos

  • Ley de Senos: $\frac{a}{\sin(A)} = \frac{b}{\sin(B)} = \frac{c}{\sin(C)}$
  • Ley de Cosenos: $c^2 = a^2 + b^2 - 2ab\cos(C)$
Funciones Hiperbólicas

Definiciones Exponenciales

  • $\sinh(x) = \frac{e^x - e^{-x}}{2}$
  • $\cosh(x) = \frac{e^x + e^{-x}}{2}$
  • $\tanh(x) = \frac{\sinh(x)}{\cosh(x)} = \frac{e^x - e^{-x}}{e^x + e^{-x}}$

Identidades Hiperbólicas

  • $\cosh^2(x) - \sinh^2(x) = 1$
  • $1 - \tanh^2(x) = \text{sech}^2(x)$
  • $\coth^2(x) - 1 = \text{csch}^2(x)$
  • $\sinh(x \pm y) = \sinh(x)\cosh(y) \pm \cosh(x)\sinh(y)$
  • $\cosh(x \pm y) = \cosh(x)\cosh(y) \pm \sinh(x)\sinh(y)$

Funciones Hiperbólicas Inversas (como logaritmos)

  • $\text{arsinh}(x) = \ln(x + \sqrt{x^2+1})$
  • $\text{arcosh}(x) = \ln(x + \sqrt{x^2-1})$ para $x \ge 1$
  • $\text{artanh}(x) = \frac{1}{2}\ln\left(\frac{1+x}{1-x}\right)$ para $|x| < 1$
Cálculo Diferencial e Integral

Reglas de Derivación

  • Regla de la potencia: $\frac{d}{dx}(x^n) = nx^{n-1}$
  • Regla del producto: $(f \cdot g)' = f'g + fg'$
  • Regla del cociente: $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
  • Regla de la cadena: $(f(g(x)))' = f'(g(x)) \cdot g'(x)$

Series y Transformadas

Series de Taylor y Maclaurin

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$$
  • $e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \dots$
  • $\sin(x) = \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \dots$
  • $\cos(x) = \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \dots$
  • $\ln(1+x) = \sum_{n=1}^{\infty} (-1)^{n+1} \frac{x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots$
